方法名 说明
this.getStoreState([parent:string])
parent：父节点input.attr('linkage')
有：获取父节点为该值的所有选中项
无：获取已配置的所有input的选中项
this.clearStoreState([parent:string, [child:string]])
parent：父节点input.attr('linkage')
有：清空父节点为该值的所有选中项
无：清空已配置的所有input的选中项

child：子节点input.attr('value')
有：清楚选项input.attr('value') == child && input.attr('linkage-parent') == parent
