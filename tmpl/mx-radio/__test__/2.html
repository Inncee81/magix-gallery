使用说明：
建议使用 <mx-radio /> 替换原生 <input type="radio" />
mx-radio对常用功能进行了封装，使用上与原生无差别；主要差别在动效和禁用样式上
mx-radio普通状态： {{each list as item}} {{/each}} 当前选中：{{=selected1}}
mx-radio禁用状态： {{each list as item}} {{/each}}
原生radio普通状态： {{each list as item}} {{/each}} 当前选中：{{=selected3}}
原生radio禁用状态： {{each list as item}} {{/each}}
HTML Code
{{!text1}}  
{{each list as item}}
<label class="mr30">
    <mx-radio 
        value="{{=item.value}}" 
        name="{{=viewId}}_radios" 
        checked="{{=(item.value == selected)}}"
        mx-change="change({value: '{{=item.value}}'})"/>
    {{=item.text}}
</label>
{{/each}}
